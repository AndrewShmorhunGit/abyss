## Getting Started

Clone repository

```bash
git clone git@github.com:AndrewShmorhunGit/abyss.git
```

Install deps

```bash
npm i
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
